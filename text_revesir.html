How to use:

  1. Enter or paste the text you want to reverse in the input box above
  2. Select your preferred reverse mode:
    • Entire Text - Reverses the complete text (e.g., "hello world" → "dlrow olleh")
    • Each Word - Reverses each word individually (e.g., "hello world" → "olleh dlrow")
    • Each Line - Reverses each line individually, preserving line breaks
  3. Click the "Reverse Text" button to see the results
  4. Use "Copy Output" to copy the reversed text to your clipboard
  5. Click "Clear" to reset both input and output fields
